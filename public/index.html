Usage Documentation

1. Authentication

Authenticate with GitHub to grant GitWeaver scoped access to your repositories. We use OAuth 2.0 to ensure your credentials remain private and secure.

2. Managing Repositories

The Dashboard provides a high-level overview of your active projects. You can search by name or filter by primary programming language.

3. Creating Projects

Use the Create Repository view to provision new projects. Selecting a template (.gitignore or License) will automatically initialize the repository with an initial commit.

4. Local Project Integration (VS Code)

To link an existing local project to your new repository:

  1. Open your project folder in VS Code.
  2. Open the integrated terminal (Ctrl + ` or Terminal > New Terminal).
  3. Copy the Remote Configuration Commands from the repository card on your dashboard.
  4. Paste and execute them in the terminal. This establishes the link between your local files and your GitHub account.

5. Direct Editing & Publishing

The Quick Editor allows for rapid changes to repository content without a local setup. Provide the file path and owner to load or update specific files directly on GitHub.
